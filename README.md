# Solution-assignment-feedforward-and-backPropagation

This implementation builds a small neural network from scratch using:

- 2 input neurons
- 2 hidden neurons
- 2 output neurons
- tanh activation function
- Random weight initialization in [-0.5, 0.5]
- Bias values:
  - Hidden layer: 0.5
  - Output layer: 0.7

The network is trained using backpropagation and gradient descent.

The final output is printed after training.
